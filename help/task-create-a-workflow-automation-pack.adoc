= 创建 WFA 工作流软件包
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在 OnCommand Workflow Automation （ WFA ）上创建工作流软件包，以满足存储自动化和集成要求。

.步骤
. 通过 Web 浏览器登录到 * WFA* 窗口。
. 单击 * 内容管理 * > * 包 * 。
. 单击 * 新软件包 * 图标。
. 在 * 新软件包 * 对话框中，为 * 名称 * ， * 作者 * ， * 版本 * 和 * 问题描述 * 字段输入值。
. 单击 * 保存 * 。
. 验证是否已在 * 包 * 窗口中创建新软件包。
