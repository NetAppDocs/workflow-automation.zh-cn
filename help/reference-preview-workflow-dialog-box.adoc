= 预览工作流对话框
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
通过 " 预览工作流 " 对话框，您可以为与工作流关联的用户输入指定值，然后使用指定值预览该工作流的执行情况。



== 用户输入值

" 预览工作流 " 对话框显示与选定工作流关联的用户输入，并可用于设置用户输入的值。



== 命令按钮

* * 预览 *
+
打开关联工作流的 " 监控 " 窗口。

* * 取消 *
+
关闭对话框。
